对现实的一些思考

发表于 2019-06-05 | 更新于 2019-06-07 | 分类于 笔记

人类目前所建立的物理理论体系，其最根本的依据是人类对现象的观察和归纳。
Is that reality?
这个问题无法被回答，因为它本身没有任何意义。实际上，我们甚至没有对”reality”的一个确切定义。
我们习惯于把感官传递给大脑的信息当成真实的，但仍存在被欺骗的风险，谁可以保证我们不是一直在被欺骗着的呢？
有个很著名的理论叫做”缸中之脑”：也许我们的大脑被某个缸装着，当然，大脑的形状也可能并不如我们所熟悉的那样。许多电极连接在我们的大脑上，导线另一头连接着运算能力极强的计算机，或者是某种能力非凡的生命。导线传递着各种电信号，来模拟各种感官，让我们获得知觉。
如果真如上面所说，我们要怎么知道我们不是处在缸中的大脑，而是一个”客观实在”的人呢？答案是没有办法。”缸中之脑”的说法是不可证伪的，但同样也无法说这就是正确的。
这样的问题没有任何意义

我们无法定义真正确切的”客观实在”。对人类来说，感官所接受的一切，包括眼睛看见的，耳朵听见的，这就是我们所说的”现实”。我们无法深究这样的”现实”是否真实，因为它对我们来说就是”真实”。
这样的说法也许有些难以理解，甚至看上去是在诡辩，我们不妨再回去思考那个物理法则的问题。
人类目前所建立的物理理论体系，其最根本的依据是人类对现象的观察和归纳。
如果这些依据从”根本”上就是”错误”的呢？也许上帝不想让我们了解”真正”的世界，于是把我们的所见所闻用一些我们无法想象的方式修改掉了。从这些”错误”的现象中建立起的物理定律，我们还能够选择相信吗？
注意，上面的讨论提到了“上帝”，我并不是有意提到与宗教相关的概念，但这样有助于我们思考，如果觉得不妥，大可想象存在一种物理过程，会影响其他某些物理过程的现象。

接着刚才的问题，这样的物理定律可靠吗？思考了一会儿，你可能会小心翼翼地说出一个不好不坏的答案：至少对人类来说是可靠的。是的，这样的物理定律对我们人类来说当然是可靠的，我们用它来解释和预测现象，建立起了物理学理论的大厦。在它们的恩泽下，我们把它们当成真理去供奉，因为它们对我们所了解到的一切现象都毫无疑问是”正确”的。
但它们实际上是”正确”的吗？

想明白了吗？这样的问题是没有任何意义的，只有当我们把”正确”定义为”符合我们所观测到的现象”时，我们才能说它们是正确的，而这个假设正是物理学所默许的，否则物理学也将没有任何意义。
