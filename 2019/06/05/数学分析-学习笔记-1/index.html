数学分析-学习笔记-1

发表于 2019-06-05 | 分类于 Mathematical_Analysis_Note

前言

经过多番思考最后还是决定开始自学《数学分析》，也就是Mathematical Analysis
我不知道能不能坚持到最后。曾经我尝试过很多次自学，学的东西很多，但大多是”技术”，准确地说是”信息技术”。按照从前的经验来说，我的自学过程往往没有计划，急于求成，很多时候都是学成半吊子便急于应用到实践。不得不说，边实践边学习的效果很不错，可缺点也很明显。我总是在一些小地方犯错误，思路不够清晰，缺乏”Big Idea”，往往视野被局限在一个小地方，最终导致计划失败。
后来发现我的知识体系十分的散乱，长时间在实践当中积累而来的经验并没有在我的大脑中形成一套系统。虽然在大多数时候遇到问题我都能想到办法解决，但这些方法都不会是最优解。
为什么我要选择《数学分析》这本书？这本书的难度可谓令人望而却步，很多大学生，包括数学系的大学生都对这本书要充满了恐惧。可我对自己的自学能力有信心，在学校里我能脱离老师的课堂而凭借书本和练习册将成绩拉到年级前列，这绝对不是偶然。这一次我想从一次实践中测试和锻炼自己的恒心、自己对知识的接受能力。
假如我能较好（且不说完全）地掌握这门课的知识，我的知识储备、学习能力绝对会有一个质的提升。此外，在未来大学当中我也能有更多余力和选择。
如果能完成这本书的学习，我会考虑按照本科生的学习轨迹向下学习更多的数学/物理知识
